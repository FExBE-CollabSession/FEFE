## 🦁 LIKELION 연합세션 세미해커톤
✴ 2025 Collaboration Session Semi-Hackathon ✴
<br><br>

## 📝 개요
약 6시간 동안 진행된 세미해커톤 대회 프로젝트입니다. <br/>
이 프로젝트는 Spring Boot 백엔드 API와 연동되는 React 기반 커뮤니티 시스템입니다. <br/> 
사용자들이 수업별로 게시글을 작성하고, 좋아요, 댓글 등의 상호작용을 할 수 있습니다.
